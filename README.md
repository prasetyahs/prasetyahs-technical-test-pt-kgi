# Laravel Product CRUD + JWT Auth + Blade + Tailwind

## Requirements
- PHP >= 8.2
- Node.js + NPM
- PostgreSQL

## Setup Project

### 1. Install dependencies backend
```bash
composer install
```

### 2. Install dependencies frontend
```bash
npm install
```
### 3. Copy environment file
```bash
cp .env.example .env
```

### 3. Generate Application key
```bash
php artisan key:generate
```
